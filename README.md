# Game Roulette UI

Готовый React + Vite + Tailwind проект.

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Загрузка на GitHub через сайт

1. Создай новый репозиторий на GitHub.
2. Нажми **Add file → Upload files**.
3. Загрузи все файлы из этого проекта.
4. Нажми **Commit changes**.

## Деплой

Проект удобно выкладывать на Vercel или Netlify.
